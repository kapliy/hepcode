\slide{Smoothing of theory uncertainties}
{
\begin{itemize}
\item Smoothing applied to theory uncertainties (PowhegPythia vs others)
\item Theory uncertainty becomes fully correlated across bins
\begin{itemize}
\item This is important for the combination!
\end{itemize}
\item Proposal: confidence interval envelope around the fitted curve
\begin{itemize}
\item Treat the size of the envelope as an uncorrelated source
\end{itemize}
\item For poly0 fit, it can be taken as 1-sigma error on fitted offset
\item For poly2, we need to keep track of correlations between pars
\begin{itemize}
\item Thanks to Max for implementing TVirtualFitter::GetConfidenceIntervals
\item Next page - a quick comparison of two techniques on poly0 fits
\end{itemize}
\end{itemize}

}

\begin{frame}[label=jul18]
\frametitle{MC-to-data reweighting}
\begin{itemize}
\item We were asked to do a cross-check: reweighting MC to data
\begin{itemize}
\item SF = (data-bg)/sig vs $p_T$ (fine bins) at detector-level
\item Separate SFs in each $\eta$ slice
\item Applied at truth level
\item Closure test: take rw-signal at reco level, unfold with nominal response matrix, and compare with rw-signal at truth level
\end{itemize}
\item In the first attempt, I estimated QCD inclusively (similar to $p_T$ stacks)
\begin{itemize}
\item The difference from that and (QCD-in-bins) is larger than the effect we are trying to correct
\end{itemize}
\item Re-made SFs, where fine-binned $p_T$ histograms were made in each $\eta$ x $p_T$ bin, and then added across $p_T$ slices.
\begin{itemize}
\item Final SFs show up to 10\% effects - the objective is to see if these effects significantly bias the result.
\end{itemize}
\end{itemize}
\end{frame}
