\slide{Notes on tau MC}
{
$\Wboson \rightarrow \tau\nu$ and $\Zboson \rightarrow \tau\tau$ MC:
\iteb
\item Powheg+Pythia - low statistics, wrong tau polarization
\item Pythia - high statistics, default for 2010, but only LO
\item Alpgen+Jimmy - high statistics, but uses LO CTEQ 6.1
\itee

Do these differences even matter?
}
