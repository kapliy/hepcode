\slide{ Comments }
{
\begin{itemize}
\item As we saw previously, MG triggers have a problem in some $\eta$ bins
\begin{itemize}
\item MG-specific SFs were used for MG plots; Muid-specific - for Muid plots
\item Buggy MG trigger matching fails to capture some inefficiencies
\end{itemize}
\item As Joao suspected, Muid chains behave differently at high $p_T$
\begin{itemize}
\item The effect is small and constrained to our last, inclusive bin ($p_{T}>50$, GeV)
\item Trigger scale factors also have one large bin to cover the 50+ GeV region
\end{itemize}
\end{itemize}
}
